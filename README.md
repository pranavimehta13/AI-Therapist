# Therapist AI 🧠

Therapist AI is a prototype conversational AI mental wellness assistant built using Streamlit, LangChain, and Ollama.

The project focuses on creating an empathetic AI chatbot capable of supportive conversations using prompt engineering and locally running LLMs.

This prototype is the foundation for a larger full-stack AI therapy platform currently in development using React, FastAPI, and advanced memory + agentic AI capabilities.

## Features

* Conversational AI therapist chatbot
* LangChain Prompt Templates
* Local LLM support with Ollama
* Streamlit chat interface
* Context-aware empathetic responses
* Lightweight and fully local setup

## Tech Stack

* Python
* Streamlit
* LangChain
* Ollama
* Llama 3.2 / Qwen / TinyLlama

## Upcoming Features

* React frontend
* FastAPI backend
* Persistent chat memory
* User authentication
* Mood tracking
* Voice interaction
* Vector database memory
* Personalized AI therapist personas
* Crisis detection & safety layers

## Goal

The aim of this project is to explore how modern LLMs, memory systems, and agentic workflows can be combined to create emotionally supportive AI applications.

⚠️ Disclaimer: This project is for educational and experimental purposes only and is not a substitute for professional mental health care.
